## 产品概述

搭建AI博客系统前端的基础架构，包括路由系统、布局组件和主题切换功能。

## 核心功能

- 路由系统：支持嵌套路由、懒加载、路由守卫
- 左侧边栏布局：Logo、导航菜单、搜索框、用户菜单
- 主题切换：亮色/暗色模式切换
- 响应式设计：桌面端左侧边栏，移动端可折叠

## 用户选择

1. 路由：createBrowserRouter + useRoutes + React.lazy
2. 布局：左侧边栏导航 + 主内容区
3. 主题：亮色 + 暗色模式
4. 导航：Logo + 导航菜单 + 搜索框 + 用户菜单下拉

## 技术栈

- 框架：Vite + React 18 + TypeScript
- 路由：react-router-dom 7.13.0
- UI组件：Tailwind CSS + shadcn/ui
- 图标：lucide-react 0.563.0
- 主题：next-themes 0.4.6

## 实现方案

### 路由架构

使用 `createBrowserRouter` + `RouterProvider` 模式，支持嵌套路由和懒加载。

**路由结构设计：**

```
/                     → 主布局
  /                   → 首页
  /blog               → 博客列表
  /blog/:id           → 文章详情
  /blog/create        → 写作页面（需登录）
  /blog/:id/edit      → 编辑页面（需登录）
  /ai-chat            → AI对话（需登录）
  /profile            → 个人主页
  /profile/edit       → 编辑资料（需登录）
  /login              → 登录页
  /register           → 注册页
```

### 布局架构

采用左侧边栏 + 主内容区的现代布局，适合AI对话和博客场景。

**布局组件层次：**

```
Layout（主布局容器）
├── Sidebar（左侧边栏）
│   ├── Logo
│   ├── Navigation（导航菜单）
│   ├── SearchBar（搜索框）
│   └── UserMenu（用户菜单）
└── MainContent（主内容区）
    └── Outlet（子路由出口）
```

### 主题切换

使用 `next-themes` 的 `ThemeProvider`，通过 CSS 变量切换亮暗色。

**实现要点：**

- 在 `main.tsx` 包裹 `ThemeProvider`
- 使用 `useTheme` Hook 获取和设置主题
- 主题切换按钮放在用户菜单中

### 路由守卫

创建 `ProtectedRoute` 组件，保护需要登录的路由。

**实现逻辑：**

- 检查 `localStorage` 中的 `access_token`
- 未登录：重定向到 `/login`
- 已登录：渲染子组件

## 架构设计

### 组件架构

```
src/
├── components/
│   └── layout/
│       ├── Layout.tsx           # 主布局容器
│       ├── Sidebar.tsx          # 左侧边栏
│       ├── SidebarHeader.tsx    # Logo区域
│       ├── SidebarNav.tsx       # 导航菜单
│       ├── SidebarSearch.tsx    # 搜索框
│       ├── SidebarFooter.tsx    # 用户菜单区域
│       ├── UserMenu.tsx         # 用户下拉菜单
│       ├── ThemeToggle.tsx      # 主题切换按钮
│       └── ProtectedRoute.tsx   # 路由守卫
├── hooks/
│   └── useAuth.ts               # 认证Hook（临时简化版）
├── routes/
│   └── index.tsx                # 路由配置
└── providers/
    └── ThemeProvider.tsx        # 主题Provider包装器
```

### 数据流

```
用户访问受保护路由
  → ProtectedRoute 检查 token
    → 无 token：重定向到 /login
    → 有 token：渲染 Layout
      → Sidebar 渲染导航
      → Outlet 渲染子页面
```

## 设计风格

采用现代简洁的设计风格，以内容为中心，适合技术博客和AI对话场景。

## 页面布局

### 左侧边栏布局

- 宽度：桌面端固定 256px，移动端可折叠
- 背景：亮色模式使用 sidebar 色彩变量，暗色模式使用深色背景
- 结构：顶部Logo → 导航菜单 → 搜索框 → 底部用户菜单

### 主内容区

- 布局：flex-1 自适应宽度
- 内边距：p-6（24px）
- 最大宽度：max-w-7xl 居中

### 导航菜单

- 样式：ghost 按钮，悬停高亮
- 图标：lucide-react 图标 + 文字标签
- 激活状态：primary 背景色

### 用户菜单

- 触发：用户头像 + 用户名
- 下拉内容：个人主页、设置、主题切换、退出登录

## 响应式设计

- 桌面端（lg及以上）：左侧边栏固定显示
- 平板/移动端：侧边栏隐藏，通过汉堡菜单按钮展开

## 交互细节

- 侧边栏折叠/展开：smooth transition
- 导航项悬停：背景色渐变
- 主题切换：无闪烁切换
- 移动端菜单：使用 Dialog 组件全屏展开
